To create a new parliament

1) close the previous parliament and goverment using 
	setup-P01/files/00-dissolve-parliament-government

2) use KfileRepla to replace from this folder setup-P01/files/
   P0-1 with P0-2
   1970 ... with 1975
   1974 ... with 1979

Of course then ... to move from P02 to P03 ... etc.
